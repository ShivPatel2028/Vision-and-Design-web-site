## 🌟 Highlights & Features

This application represents the pinnacle of modern web design, meticulously crafted to deliver a luxurious, cinematic, and highly-immersive user experience:

- **Cinematic A+++ Design**: Full-bleed hero sections, physics-based canvas particle systems, mouse-following parallax glass orbs, and luxury typography (`Playfair Display` + `Inter`).
- **Advanced Motion Design**: Smooth scroll reveals, motion-blur-inspired transitions, infinite marquee effects, and highly engaging micro-interactions powered by `framer-motion`.
- **Form-Free Direct Contact**: Designed exclusively with direct click-to-call, click-to-email, and WhatsApp links to ensure immediate, high-touch communication with the founders.
- **Dedicated Vertical Experiences**: Six uniquely designed service pages (Branding, Digital Design, Event Management, Fashion Choreography, Construction, Interior Design), each with its own tailored visual identity and cinematic storytelling.
- **Optimized Performance**: Pre-rendered static pages via Next.js App Router for instant load times and perfect Lighthouse SEO scores.

## 🚀 Getting Started

This is a Next.js project bootstrapped with `create-next-app`.

### Prerequisites
- Node.js (v18.17 or later)
- npm, yarn, pnpm, or bun

### Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ShivPatel2028/Vision-and-Design-web-site.git
   cd Vision-and-Design-web-site
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **View the application:**
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the cinematic experience.

## 📂 Project Structure

- `/app`: Next.js 15 App Router pages (Home, About, Contact, Services, Portfolio, etc.)
- `/components`: Modular UI architecture
  - `/home`: Page-specific components for the landing experience
  - `/layout`: Global layout components (Navbar, Footer)
  - `/shared`: Reusable elements (AnimatedSection, CustomCursor)
- `/lib`: Global constants, utility functions, and static data configuration
- `/styles`: Global CSS, custom glassmorphism layers, and animation keyframes

## 🌐 Deployment

The easiest way to deploy your Next.js app is to use the [Vercel Platform](https://vercel.com/new).

1. Push the code to your GitHub repository.
2. Import the project into Vercel.
3. Vercel will automatically detect Next.js and apply the correct build settings.
4. Deploy!
